
## GAME MECHANICS (CRITICAL):

1. **10x10 Grid Board** - The main playing field
2. **Block Pieces** - 3 random pieces shown at bottom. Player drags and drops them onto the board
3. **Line Clearing** - When a full row OR full column is completed, it clears with animation and awards points
4. **Combo System** - Clearing multiple lines at once gives bonus points (x2, x3, x4 multipliers)
5. **Game Over** - When none of the remaining pieces can fit anywhere on the board
6. **Scoring**:
   - Single line: 100 points
   - Each additional simultaneous line: +200 bonus
   - Combo multiplier for consecutive clears
   - Piece placement: 10 points per block

## DRAG AND DROP SYSTEM:
- Use React Native PanResponder or react-native-gesture-handler
- When user starts dragging a piece, it should:
  - Scale up slightly (1.1x)
  - Show a shadow beneath
  - Float above other elements (high zIndex)
  - Show ghost/preview on the board where it would be placed
- The ghost preview should be semi-transparent
- If placement is valid: ghost is green-tinted
- If placement is invalid: ghost is red-tinted
- On drop: if valid position, place with satisfying animation
- On drop: if invalid, animate piece back to original position
